About Garage Management System

The Garage Management System helps you manage vehicles in your garage efficiently. It allows users to add, view, and track vehicles, making it perfect for small garages or workshops.

Features

  • Add and store vehicle information
  • View a list of all vehicles
  • Simple and clean user interface
  • Connects with backend APIs for live updates

Technologies Used

  • HTML, CSS, JavaScript (Frontend)
  • Node.js, Express.js (Backend)
  • JSON for data handling

Purpose

This system was created as a lightweight, easy-to-use solution for managing vehicle records in local garages, aiming to replace manual logs with a digital interface.
